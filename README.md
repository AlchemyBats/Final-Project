# Final-Project
Run main.py
You should be able to review results without issue.
If you want to regenerate sentiment scores, you will need to download my cleaned SEC dataset here: https://drive.google.com/drive/folders/1AFNIApHXPzyrdQIJhf21PF4fEkYKq3Xv?usp=drive_link
Add the folder reports_formatted to the main project folder. You will also need the word2Vec model here: https://www.kaggle.com/datasets/leadbest/googlenewsvectorsnegative300/data
Add the .bin file to a new folder called pretrained. You will also need to download this Ollama model: https://ollama.com/library/qwen3-coder
